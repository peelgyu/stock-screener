📱 StockInto 앱 설치하기

브라우저에서 바로 설치할 수 있는 웹앱(PWA) 방식과 APK 다운로드 방식 모두 지원합니다.

📱 Android
🍎 iPhone
📦 APK

Android - 브라우저로 설치 (추천)

Google Play 없이 Chrome 브라우저에서 바로 설치 가능합니다.

1. Chrome으로 stockinto.com 접속
2. 주소창 오른쪽 ⋮ (메뉴) 버튼 탭
3. "앱 설치" 또는 "홈 화면에 추가" 선택
4. 설치 버튼 탭 → 홈 화면에 StockInto 아이콘 생성
5. 아이콘 탭해서 앱 실행 (주소창 없는 전체 화면)
💡 사이트 하단에 "📱 앱으로 설치" 버튼이 뜨면 원클릭으로 설치 가능합니다.

iPhone / iPad - Safari로 설치

iOS는 Safari 브라우저만 지원합니다 (Chrome·삼성브라우저 불가).

1. Safari로 stockinto.com 접속 (Chrome 아님!)
2. 하단 공유 버튼 (⬆) 탭
3. 목록에서 "홈 화면에 추가" 선택
4. 이름 확인 후 "추가" 탭
5. 홈 화면에 StockInto 아이콘 생성 완료
⚠ iOS는 Apple 정책으로 PWA 기능이 일부 제한됩니다 (푸시 알림 등). 기본 사용엔 지장 없음.

APK 파일 직접 다운로드

브라우저 설치가 싫으면 APK 파일을 받아서 설치할 수 있습니다.

⚠ APK 버전은 준비 중입니다. 배포 준비 완료 후 여기에 다운로드 링크가 추가됩니다.

예상 설치 절차

  1. APK 파일 다운로드
  2. 설정 → 보안 → "알 수 없는 출처 허용" 켜기
  3. 다운로드한 APK 파일 탭 → 설치
  4. 홈 화면에 StockInto 앱 생성

브라우저 설치 vs APK 비교

  • 브라우저 설치 (PWA): 즉시 가능, 자동 업데이트, 용량 작음
  • APK: 수동 업데이트 필요, 하지만 완전 독립 실행
